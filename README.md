# toy-shell

## How to Run WooShell
1. Download
	git clone https://github.com/kws6081/toy-shell
	cd /toy-shell

2. Complie C codes
	gcc -o wooshell shell.c && gcc -o info info.c && gcc -o red red.c && gcc -o undocolor undocolor.c 

3. Run wooshell
	./wooshell

4. Run commands
	Every commands in /bin.
	info
	red
	undocolor
	clear
	exit
